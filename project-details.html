Online Store Web Application System For Grocery Shops

Project Description

โปรเจกต์นี้พัฒนาในรูปแบบ Full-stack Web Application โดยใช้ React HTML CSS สำหรับส่วน Frontend และ Node.js ร่วมกับ Express.js สำหรับส่วน Backend ส่วนฐานข้อมูลใช้ Firebase Firestore ซึ่งเป็น NoSQL Database แบบ Real-time สำหรับจัดเก็บและจัดการข้อมูลสินค้า ผู้ใช้ และคำสั่งซื้อทั้งหมด นอกจากนี้ยังมีการใช้ Docker สำหรับทำ Containerization เพื่อช่วยให้การ Deploy ระบบทำได้ง่ายขึ้น และรองรับการขยายระบบ (Scalability) ได้อย่างมีประสิทธิภาพ

Features

  • ระบบสมัครสมาชิก เข้าสู่ระบบ และจัดการโปรไฟล์ผู้ใช้งาน
  • การเลือกดูสินค้าแยกตามหมวดหมู่ เช่น อาหาร เครื่องดื่ม ของใช้ในบ้าน และยา
  • ระบบตะกร้าสินค้าและการจัดการคำสั่งซื้อ พร้อมการคำนวณราคาและส่วนลด
  • ระบบจัดการคลังสินค้าแบบเรียลไทม์ พร้อมการติดตามจำนวนสต็อก
  • ระบบแจ้งเตือนสำหรับทั้งผู้ใช้งานและผู้ดูแลร้าน
  • ระบบถูกจัดการและ Deploy ด้วย Docker เพื่อให้การติดตั้งและใช้งานมีความสะดวกและสม่ำเสมอในทุกสภาพแวดล้อม
